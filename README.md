# 系統助手

## 系統概述

系统助手是一个智能技术支持系统，专注于提供：
- 实时天气查询和天气警告监控
- 智能代码编写和优化建议
- 系统管理和监控（Agents、数据库、Docker）
- 技术咨询、建议和优化建议
- 数据查询、统计、分析和报告生成

## 主要功能

- ✅ **天气查询** - 香港天文台天气数据、预报、警告
- ✅ **天气监控** - 自动监控天气变化，检测酷热、暴雨、强风、台风警告
- ✅ **代码助手** - Python、JavaScript、Bug 修复、代码优化
- ✅ **系统管理** - 配置、监控、优化、重启
- ✅ **技术咨询** - 技术问题解答、建议、优化建议
- ✅ **数据分析** - 数据查询、统计、分析、报告生成

## 系统架构

- **主程序** - 任务分发、智能路由、系统协调
- **对话助手** - 信息查询、问答、聊天
- **工程助手** - 代码编写、编程、算法
- **系统管理员** - 系统配置、监控、优化
- **天气助手** - 天气查询、天气监控、警告

## 快速开始

- 查询天气：`现在几度？`
- 代码编写：`帮我写个 Python 脚本`
- 系统配置：`系统配置在哪里？`
- 技术咨询：`你有什么建议吗？`

## 系统状态

- **系统版本**：v2.0
- **Agents**：4 個（主程序、对话助手、工程助手、系统管理员、天气助手）
- **数据库**：PostgreSQL（天气数据、对话数据、会话数据、优化记录）
- **监控**：Heartbeat 监控、警报通知、自动重启
- **自动清理**：每天凌晨 2 点自动清理旧数据

## 技术特色

- **智能路由** - 自动分发任务到最合适的 Agent
- **话题识别** - 自动识别用户查询的主题（10 种话题类型）
- **Session 管理** - 智能切换 Session，减少 Token 消耗
- **自动优化** - 每日自动优化系统性能和准确度
- **健康监控** - 实时监控所有 Agents 和系统组件
